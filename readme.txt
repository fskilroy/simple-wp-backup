A simple WordPress backup plugin that does nothing (yet).

== Description ==

This is a simple WordPress backup plugin that currently does nothing. It serves as a basic plugin structure that can be extended with actual backup functionality in the future.

Features:
* Proper WordPress plugin structure
* GPL licensed
* Ready for future development

== Installation ==

1. Upload the `simple-wp-backup` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin is now active but doesn't do anything yet

== Frequently Asked Questions ==

= What does this plugin do? =

Currently, nothing. It's a basic plugin structure ready for future development.

= Is this plugin safe to use? =

Yes, since it doesn't do anything, it won't interfere with your site.
